# ByteCase Acquire Dependencies

Runtime:

- Python standard library
- Tkinter / Tcl-Tk, distributed with most Python installations
- openpyxl, MIT License, used for XLSX tracking/statistics exports
- python-docx, MIT License, used for DOCX report export

No new runtime dependency was added for v0.9.2.

PDF export is intentionally parked until a license-safe and reliable PDF path is selected. The current release continues to generate DOCX, TXT, JSON, and XLSX outputs.
